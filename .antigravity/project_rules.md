# Project Rules: TAKNET-PS ADS-B Feeder

These rules apply to all coding updates within this repository.

1.  **Synchronized Versioning**: All coding updates MUST update the version string in all locations where it is referenced.
    - `README.md`
    - `install/install.sh`
    - `version.json`
    - `VERSION`
2.  **Release Documentation**: The `version.json` file's `release_notes` MUST be updated to describe the changes in each version.
3.  **Default Priority**: Set `update_priority` to `3` in `version.json` unless explicitly instructed otherwise.
4.  **Version Control**: All changes MUST be pushed to the git repository for this project only.
5.  **Aesthetics**: As per system instructions, prioritize premium design and WOW factor for all UI updates.
